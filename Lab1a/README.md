# Lab01a-NumbersGame

## Exception-Handling

*Author: Lesley Rivera*

----

## Description
This is a C# console application that will create a game that takes user input, manipulates data, and utilizes
error handling, by following the given specifications.
At the end of the game the user receives their total score.

---

### Getting Started
Clone this repository to your local machine.

```
$ git clone [repo clone url here]
```

### To run the program from Visual Studio:
Select ```File``` -> ```Open``` -> ```Project/Solution```

Next navigate to the location you cloned the Repository.

Double click on the ```Lab01-Numbers Game``` directory.

Then select and open ```Lab1a.sln```
